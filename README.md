# BUJKT

BUJKT is an unpublished Chrome extension for organizing Objkt bookmarks, sorting listed artwork by price, and helping collectors quickly navigate to collectible items.

If you want to collect on a BUJKT without blowing your budget, this extension helps you see the cheapest gems first.

## Features

- Sort Objkt bookmarks by price so you can spot budget-friendly buys
- Highlight collectible items and make browsing easier
- Keeps your art hunt fast, fun, and slightly less wallet-scary

> Tip: On Objkt bookmarks, keep clicking on the **Load more** button until all items are loaded so the extension can sort all loaded items. Objkt only shows a few bookmarks at a time by default.

## Installation

This extension is not published on the Chrome Web Store.

To install:

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** in the top-right corner
3. Click **Load unpacked**
4. Select the folder containing this extension

> Note: This extension is currently unpublished and must be loaded unpacked in developer mode.
